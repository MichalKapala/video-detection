Video Detection - Dokumentacja

Repozytorium

Repozytorium: https://github.com/MichalKapala/video-detection

Opis działania aplikacji

Aplikacja Video Detection pozwala użytkownikowi na załadowanie pliku wideo w formacie .avi, na którym zostaną rozpoznani ludzie. Model machine learningowy działa w trakcie odtwarzania wideo, a detekcje, które generuje służą do oznaczenia rozpoznanych ludzi na wideo. Detekcja na wideo reprezentowana jest zielonym prostokątem, w którym znajduje się rozpoznany obiekt.

Szczegółowe opcje

  • Aplikacja pozwala na załadowanie pliku wideo w formacie .avi
  • Aplikacja pozwala na włączanie lub wyłączanie procesowania wideo w celu generowania detekcji, tj. w trakcie odtwarzania wideo istnieje możliwość wyłączenia lub włączenia pracy modelu do rozpoznawania ludzi za pomocą dedykowanego checkboxa
  • Aplikacja pozwala na zatrzymywanie i wznawianie wideo
  • Aplikacja pozwala na nawigację po wideo za pomocą slidera (paska postępu wideo)
  • Aplikacja pozwala na zapis wideo z zaznaczonymi detekcjami
  • Aplikacja pozwala na zapis wygenerowanych detekcji do pliku w formacie JSON

Organizacja repozytorium

Aplikacja podzielona jest na dwa główne moduły: frontendowy, odpowiadający za interfejs użytkownika oraz backendowy, odpowiadający za logikę działania aplikacji i modelu rozpoznawania obiektów i generowania detekcji. Kod frontendowy umieszczony jest w katalogu View, natomiast moduły backendowe w katalogach FrameProcessor oraz VideoProcessor. W katalogu Resources umieszczony jest plik wideo w formacie .avi, który służył do testowania działania aplikacji. W głównym katalogu znajduje się plik main.py, który służy do uruchomienia całej aplikacji. W tym samym katalogu znajduje się również plik requirements.txt służący do zainstalowania modułów potrzebnych do uruchomienia aplikacji.

Wymagania wstępne przed uruchomieniem aplikacji

Python w wersji 3.10.8

Instrukcja uruchomienia aplikacji

  1. Sklonować repozytorium za pomocą komendy: git clone git@github.com:MichalKapala/video-detection.git
  2. W głównym katalogu aplikacji należy utworzyć środowisko wirtualne Pythona venv za pomocą komendy:
    python -m venv venv
  3. Aktywowanie środowiska wirtualnego za pomocą komendy:
    source ./venv/bin/activate
  4. Zainstalowanie wymaganych modułów za pomocą komendy:
    pip install -r requirements.txt
  5. Po instalacji modułów można uruchomić aplikację za pomocą komendy:
    python main.py
